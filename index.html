What is PrepFlight?

PrepFlight is a mobile app (iOS + Android) that lets you practice the six core cognitive exercises used in the DLR (Deutsches Zentrum für Luft- und Raumfahrt) pilot aptitude test — the standard entry exam for Lufthansa, airberlin (historical), and several other European airline cadet programs. Each game ports the timing, scoring, and difficulty progression of the original exam, including a Realistic mode that simulates the live test pacing.

No account required. Local scoring. Free tier includes every game on the easiest setting; paid tiers unlock all difficulties, all settings, and Realistic mode.

The Six Games

RMS — Running Memory Span

Listen to 8–30 spoken digits, then type them back in reverse order. Tests working memory and auditory recall.

ROT — Cube Rotation

A cube is announced with a starting face up. 10–16 audio rotation commands follow. Identify which face ends on top. Tests spatial reasoning under audio-only input.

SKT — Concentration

Triangles flash one at a time, each with a colour, rotation, and dot count. For every triangle, compare two random properties to the previous one. Tests sustained attention.

MEK — Visual Memory

Memorise 20 image-number pairs presented in shrinking groups (4-4-2-2-2-2-1-1-1-1), then recall the number paired with each image. Tests visual-associative memory.

VLR — Spatial Orientation

An arrow points left or right. A 12-segment path is then drawn. Count the turns matching the arrow's direction. Tests spatial tracking and counting under load.

OWT — Perceptual Speed

Eight clock-style instruments appear (circle/rectangle, black/white). Identify the four matching the given criterion and recall their numbers. Tests visual scanning speed.
